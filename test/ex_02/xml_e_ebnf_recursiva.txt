literal     = +qi::int_;
variable    = lexeme [ qi::alpha >> *qi::alnum ];
term        =  literal 
           | variable 
           | (qi::lit('(') > expression >> ')');

factor      = term >> *(qi::char_("+-") >> term);
expression  = factor >> *(qi::char_("*/%") >> term);


Notar que "expression" faz referencia circular para "factor" e "term"
