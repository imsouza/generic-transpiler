Este compilador deve ser capaz de:
1 - identificar variáveis: i=10;
2 - identificar expressões do tipo if/else (formato: if(<expressão>){}else<if..>{}
3 - executar o laço de repetição for(for i in range(5)){}

exemplo:

valor=3;
for(i in range (10))
{
    if(i<valor)
   {
       print("menor que ",valor);
   }
   else if (i==valor)
   {
       print("igual a ", valor);
   }
   else
  {
     print("maior que ",valor);
  }
}
